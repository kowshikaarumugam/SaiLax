How to Deploy & Install

Follow these basic guidelines to run, sync, and deploy your personalized OwnUp app.

1

Download the APK

Download the compiled OwnUp debug APK file directly onto your Android smartphone. Click the primary action button below to trigger the package downloader.

2

Enable Unknown Sources

Since OwnUp is currently compiled directly from development builds, navigate to your Android device's Settings > Apps > Special App Access > Install Unknown Apps to authorize setup.

3

Optional Firebase Config

To enable Firestore cloud backup capabilities on your own builds, simply put your custom google-services.json inside the /app directory and build the project!
